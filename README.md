# Backend meets Frontend: Write technical UI tests object-oriented or functional
When the first UI tests are running, it's time to select elements of the web application and write technical tests. This can be object-oriented or functional. The Selenium essay TestBench helps to make development life easier.


## How to start
To play with this you only have to start the main - method from the class  ```org.rapidpm.vaadin.helloworld.server.Main``` and 
go with your browser to the address: [http://localhost:8080/](http://localhost:8080/)

Ok, you got your first Vaadin "Hello World".
